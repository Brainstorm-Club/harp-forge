# public/pdf/

Qui va il **template AcroForm della scheda HARP italiana** (blank, campi
editabili) usato per l'export PDF (Epic 3). Da fornire: una tantum si mappano
i campi in `src/utils/pdfFieldMapping.ts`.

Le schede reali compilate (es. PAU-LA500) restano in `assets/` e NON sono
pubblicate (vedi `.gitignore`).
